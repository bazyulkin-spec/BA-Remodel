# BA-Remodel · Android

Планировщик раскладки плитки: комната → узор → расчёт материалов → PDF-отчёт.
Kotlin + Jetpack Compose, офлайн, без сторонних сервисов.
_BA-Remodel · Inspired by Alexander Baziulkin._

## Сборка

1. Android Studio **Ladybug** или новее, JDK **17** (Settings → Build Tools → Gradle → Gradle JDK).
2. `File → Open…` → выбрать папку `BA-Remodel` → дождаться Gradle Sync
   (первый раз качает AGP 8.7.3, Kotlin 2.1.0, Compose BOM 2024.12.01 — нужен интернет).
3. Тесты движка: `./gradlew :core:test` (Windows: `gradlew.bat :core:test`).
4. APK: `./gradlew :app:assembleDebug` → `app/build/outputs/apk/debug/app-debug.apk`,
   либо кнопка ▶ Run на подключённом телефоне (Android 8.0+).

Если Gradle ругается на wrapper — `File → Sync Project with Gradle Files`,
Android Studio пересоздаст `gradle/wrapper` сама.

## Что умеет

- Комната: прямоугольник по размерам, L-форма, произвольный контур —
  тянуть углы, добавлять углы кнопкой «+» на стене, удалять лишние; снап к сантиметру и к стенам.
- Вырезы (колонна, короб, ниша): добавить, двигать, тянуть за угол, задать точные размеры.
- Узоры: сетка, вразбежку ½ и ⅓, ёлочка; поворот 0–90°, сдвиг узора пальцем.
- Плитка: пресеты и точные размеры, ширина шва, палитра цветов, разнотон, фото своей плитки.
- Расчёт: площадь, периметр, целые плитки, подрезка, всего, запас 5/10/15 %, сколько купить (шт и м²).
- Советы: сравнение 7 вариантов раскладки, топ-3 по расходу и числу подрезок, применение в один тап.
- Проекты: сохранение в память телефона, список, загрузка, удаление.
- Отчёт: карточки на экране + PDF через системный «Поделиться» (карта подрезки внутри).
- Интерфейс: русский и английский по системной локали.

## Если нет Android Studio и Android-телефона

**1. Версия для браузера — `BA-Remodel.html`.** Один файл, всё внутри (React вшит),
интернет не нужен. Открывается на компьютере и на телефоне (в т.ч. iPhone):
отправить файл себе в мессенджер/почту → открыть в браузере.
На iPhone: «Поделиться» → «На экран «Домой»» — запускается как приложение, без адресной строки.
Функциональность та же, что в Android-версии, кроме сохранения проектов между сессиями
и системного шаринга PDF (отчёт печатается через «Печать → Сохранить в PDF»).

**2. APK без установки чего-либо — сборка в облаке (GitHub Actions).**
В проекте уже лежит `.github/workflows/android.yml`.
1. Создать бесплатный аккаунт на github.com → `New repository` (можно приватный).
2. `Add file → Upload files` → перетащить содержимое папки `BA-Remodel`
   (или `git push`, если git есть) → Commit.
3. Вкладка **Actions** → workflow «Android APK» запустится сам
   (если нет — кнопка `Run workflow`). Сборка ~4–6 минут.
4. Открыть завершившийся запуск → раздел **Artifacts** → скачать `BA-Remodel-debug-apk`.
   Там же `core-test-report` — отчёт по тестам движка.
APK можно поставить на любой Android 8.0+ (в настройках разрешить установку из этого источника).

**3. Собрать локально без Android Studio** — нужен только JDK 17 и Android command-line tools:
`sdkmanager "platforms;android-35" "build-tools;35.0.0"`, затем `./gradlew :app:assembleDebug`.
